# Dungeons

> Dungeons is so evil

Features:

- Generate random maze per game.
- Press `r` for auto-play.

## How to play

Press `Enter` to execute a command.

### Commands

- Movement: `w`, `s`, `a`, `d`
- Attack: `f`
- Pick Item: `e`
